About this project

This dummy application is a part of a project activity based on topics treated by Infrastructures for Cloud Computing and Big Data course teached by Prof. Antonio Corradi at University of Bologna.
The project activity is the result of a collaboration with the company Imola Informatica S.p.a. set in Imola, Italy.
This application has been developed in order to be run, for testing, on environments with containerization features such as Docker, Kubernetes and Openshift.
